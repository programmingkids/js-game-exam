## 製作テスト
製作物を作成する課題です。課題の要件に従ってゲームを構成します。  

### 縦スクロールタンクゲーム
縦スクロールのタンクゲームです。プレイヤー（タンク）と敵タンクが登場します。  
プレイヤーによる攻撃、敵による攻撃が発生します。  
スコアとライフが存在します。  
敵を倒すとスコアアップします。  
敵の攻撃を受けるとライフが減少します。  
ライフが0になるとゲームオーバーになります。  

* プレイヤーの攻撃処理作成
* 敵グループ作成
* 敵グループの攻撃処理
* プレイヤーが敵を倒す
* スコアアップ
* ライフ
* アイテムグループ作成
* ゲームオーバーの仕組み
